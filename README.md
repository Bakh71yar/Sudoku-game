# Sudoku-game
Morning Sudoku Arena — это игровой тренажер логики, где классическое Судоку превращается в ежедневную тренировку мышления, скорости и концентрации.

Приложение создано для школьников, студентов, взрослых, любителей brain training и игроков, которым нравится соревноваться. Пользователь не просто решает головоломки — он видит свой рост через streak, XP, статистику, рейтинг, чистые победы без подсказок и ежедневный прогресс.

Главное отличие проекта - AI Coach. В отличие от обычной подсказки, AI Coach не просто показывает правильный ответ, а объясняет логику следующего хода. Пользователь может выбрать уровень помощи: маленький намек, объяснение логики или следующий ход. Благодаря этому приложение становится не механической игрой, а персональным тренером по Судоку.

Ключевые функции:
- Ежедневное испытание с ежедневным рейтингом
- 5 уровней сложности: легкий, средний, сложный, экспертный, мастерский
- AI Coach с 3 режимами помощи: маленький намек, объяснение логики, следующий ход
- Глобальный и ежедневный лидерборд
- Профиль пользователя со streak, XP, статистикой, прогрессом и чистыми победами
- Система открытия Pro: пригласи 3 друзей, чтобы разблокировать Pro-функции
- Pro Trial: реши 10 уровней без подсказок, чтобы получить временный доступ к Pro
- Pro-функции: аватарки, уровни Expert/Master, продвинутый AI Coach, расширенная статистика
- Светлая и темная тема

Morning Sudoku Arena превращает Судоку в ежедневную арену логики, прогресса и соревнования.
